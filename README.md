# Android Assignment Question 3 – Weather Information App

## 📌 Overview

This project is developed as part of the **CSE227 – Advanced Android App Development** course assignment.

The application demonstrates how to consume a **REST API** using **Retrofit** to fetch and display live weather information based on the city entered by the user.

---

## 🎯 Features

- 🌤️ Live weather information
- 🔍 Search weather by city
- 🌡️ Current temperature
- ☁️ Weather condition
- 💧 Humidity
- 💨 Wind speed
- 🌍 Country information
- ⏳ Progress indicator while loading
- ❌ Error handling for invalid cities
- 📱 Clean Material Design interface

---

## 🛠️ Technologies Used

- Kotlin
- Android Studio
- Retrofit
- Gson Converter
- REST API
- WeatherAPI.com
- Material Design Components

---

## 📡 API Used

**WeatherAPI.com**

The application retrieves real-time weather information using the WeatherAPI REST service.

## 🚀 How to Run

1. Clone the repository.
2. Open the project in Android Studio.
3. Sync Gradle files.
4. Add your WeatherAPI.com API key in `MainActivity.kt`.
5. Build and run the application.
